API Documentation

Access real-time ERCOT settlement prices and capacity monitoring data

üöÄ Getting Started

The ERCOT Analytics API provides access to two main data sources:

Settlement Point Prices

Real-time electricity pricing data for all ERCOT hubs and load zones, updated every 15 minutes.

Capacity Monitor Data

Live grid capacity and ancillary services data, updated every minute for real-time monitoring.

Settlement Point Prices

Access real-time ERCOT settlement point pricing data

GET
/api/v1/settlement-prices

Query Parameters:

  • start_date (optional) - Start date (YYYY-MM-DD)
  • end_date (optional) - End date (YYYY-MM-DD)
  • hub (optional) - Filter by hub: houston, north, south, west, busavg, hubavg
  • limit (optional) - Max records (default: 1000, max: 10000)
Example Request:
GET /api/v1/settlement-prices?hub=houston&limit=100
Available Hubs:
‚Ä¢ hb_busavg, hb_houston, hb_hubavg, hb_north, hb_south, hb_west - lz_aen, lz_cps, lz_houston, lz_lcra, lz_north, lz_south, lz_west

Capacity Monitor Data

Access real-time ERCOT capacity and ancillary services data

GET
/api/v1/capacity-monitor

Query Parameters:

  • start_date (optional) - Start date (YYYY-MM-DD)
  • end_date (optional) - End date (YYYY-MM-DD)
  • category (optional) - Filter by category
  • subcategory (optional) - Filter by subcategory
  • limit (optional) - Max records (default: 1000, max: 10000)
Example Request:
GET /api/v1/capacity-monitor?category=EMR,%20OUT,%20and%20OUTL%20Capacity%20(MW)

Authentication

All API requests require authentication using your API key and secret:

Headers Required:
X-API-Key: your_api_key_here X-API-Secret: your_api_secret_here

Code Examples

Ready-to-use code snippets in multiple programming languages

Python Example

import requests # Your API credentials api_key = "your_api_key_here" api_secret = "your_api_secret_here" # Headers headers = { "X-API-Key": api_key, "X-API-Secret": api_secret } # Get latest settlement prices for Houston hub response = requests.get( "http://52.4.166.16/api/v1/settlement-prices", headers=headers, params={"hub": "houston", "limit": 100} ) data = response.json() print(f"Retrieved {data['count']} records") for record in data['data'][:5]: print(f"{record['timestamp']}: ${record['hb_houston']}/MWh")

JavaScript Example

const apiKey = 'your_api_key_here'; const apiSecret = 'your_api_secret_here'; fetch('http://52.4.166.16/api/v1/settlement-prices?hub=houston&limit=100', { headers: { 'X-API-Key': apiKey, 'X-API-Secret': apiSecret } }) .then(response => response.json()) .then(data => { console.log(`Retrieved ${data.count} records`); data.data.slice(0, 5).forEach(record => { console.log(`${record.timestamp}: $${record.hb_houston}/MWh`); }); });
